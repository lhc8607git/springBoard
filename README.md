## 👋 소개
 이 프로젝트는 스프링 기반의 MVC 구조로 설계하여, 로그인, 회원가입, 게시판을 개발하였습니다.

## ⚙️ 설계 및 개발 내용 :
- 회원 관리 (회원가입/로그인/로그아웃) 설계 및 개발
- 게시글 관리 (게시글 CRUD, 댓글 CRUD, 첨부파일 등록/삭제) 설계 및 개발
- 게시글 검색, Pagination, Indexing
- 불필요한 데이터 주기적으로 삭제하는 Batch 개발
